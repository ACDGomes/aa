# Ferramenta Interativa de Estudo

Este é um projeto de conclusão do curso "Unidade de Página web: desenvolvendo uma ferramenta interativa de estudo" da plataforma Alura.

## Como Usar

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.

## Estrutura do Projeto

- `index.html`: Página inicial
- `about.html`: Informações sobre a ferramenta
- `quiz.html`: Quiz interativo
- `styles/style.css`: Estilos da página
- `scripts/main.js`: Script do quiz
